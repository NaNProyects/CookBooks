Usuarios [U]
========
U1) Registrarse en la aplicación
U2) Iniciar sesión
U3) Cerrar sesión
U4) Recuperar contraseña
U5) Modificar perfil
U6) Eliminar cuenta

Libros (usuarios) [LU]
=============
LU1) Buscar un libro
LU2) Ver perfil de un libro
LU3) Hojear un libro //depende de LU2)
LU4) Enviar un formulario de contacto
LU5) Ver historial de compras
LU6) Ver la página de ayuda


Carrito [C]
=======
C1) Agregar un libro al carrito
C2) Quitar un libro del carrito
C3) Ver el contenido del carrito
C4) Confirmar carrito (comprar) //BOOM
C5) Validar tarjeta //depende de C4)

Administración [A]
============
A1) Ver lista de libros
A2) Agregar un libro
A3) Modificar datos de un libro
A4) Eliminar un libro
A5) Agregar un autor
A6) Eliminar un autor
A7) Ver lista de pedidos
A8) Marcar un pedido como enviado
A9) Ver estadísticas de ventas
